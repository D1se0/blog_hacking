# 🛡️ Blog de Hacking Ético & Writeups

Bienvenido a mi repositorio personal dedicado al hacking ético, donde documento mi aprendizaje, práctica y evolución en el mundo de la ciberseguridad.

Este espacio reúne writeups, apuntes técnicos, cheatsheets y laboratorios prácticos enfocados en mejorar habilidades ofensivas y defensivas.

## 📌 Contenido del repositorio

### 🧠 Writeups de plataformas `CTF`

Aquí encontrarás resoluciones detalladas de máquinas y retos de distintas plataformas:

- 🔐 Hack The Box (`HTB`)
  - Máquinas activas → 🔒 Protegidas con contraseña
  - Máquinas retiradas → 🌍 Disponibles públicamente
- 🐳 `Dockerlabs`
- 🎯 `PicoCTF`
- 🧩 Otros retos y plataformas

Cada writeup incluye:

- Metodología paso a paso
- Herramientas utilizadas
- Explicación de vulnerabilidades
- Alternativas de explotación (cuando aplica)
